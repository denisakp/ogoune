# Ogoune Cloud

Don't want to run Ogoune yourself? **Ogoune Cloud** is the managed version — the exact same code, hosted and operated by us.

## What you get

- No infrastructure to run — we handle Postgres, Redis, upgrades, and backups
- Same features as self-hosted (Community + Enterprise)
- Managed scaling and uptime

## Self-host vs Cloud

| | Self-host | Cloud |
|---|---|---|
| Infrastructure | You run it | We run it |
| Cost | Free (CE) / license (EE) | Subscription |
| Upgrades | Manual | Automatic |
| Data location | Yours | Managed |

Nothing in Cloud is closed or secret — it's the open-source app, managed.
